Git 사용법: Git이란 무엇인가? & 설치 및 설정

1. Git이란 무엇인가?

Git은 분산 버전 관리 시스템으로, 소스 코드의 변경 이력을 관리하고 협업을 가능하게 하는 도구입니다.

Git의 주요 특징

  • 분산형 구조: 중앙 서버 없이도 각 클라이언트가 전체 데이터를 관리.
  • 효율성: 파일의 전체 스냅샷 대신 변경 사항만 저장.
  • 신뢰성: SHA-1 해시를 사용해 데이터 무결성을 보장.
  • 협업 지원: 여러 개발자가 동시에 작업 가능.

Git의 장점

  1. 백업 기능: 작업 내역을 항상 저장소에 보관.
  2. 협업 강화: 브랜치를 통해 각자 독립적으로 작업 가능.
  3. 오픈소스: 무료로 사용 가능하며 커뮤니티가 활발.

2. Git 설치 및 설정

Git 설치

  • Windows

    1. Git 공식 웹사이트에서 설치 파일 다운로드.
    2. 설치 중 기본 옵션 선택 추천.
    3. 설치 완료 후 Git Bash 실행.

  • Mac

    1. Homebrew로 설치:
      brew install git

  • Linux

    1. Debian/Ubuntu:

      sudo apt update
sudo apt install git

    2. CentOS/Fedora:

      sudo dnf install git

Git 초기 설정

Git 설치 후, 사용자 정보와 에디터를 설정합니다.

  1. 사용자 이름 설정:

    git config --global user.name "사용자이름"

  2. 사용자 이메일 설정:

    git config --global user.email "사용자이메일@example.com"

  3. 기본 에디터 설정 (예: VS Code):

    git config --global core.editor "code --wait"

  4. 설정 확인:

    git config --list

Git 명령어를 사용할 준비가 완료되었습니다! 😊
