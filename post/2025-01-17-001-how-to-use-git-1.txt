f:Tbfa,<h1 id="git-사용법-git이란-무엇인가-설치-및-설정" tabindex="-1">Git 사용법: Git이란 무엇인가? &amp; 설치 및 설정</h1>
<h2 id="1-git이란-무엇인가" tabindex="-1">1. Git이란 무엇인가?</h2>
<p>Git은 <strong>분산 버전 관리 시스템</strong>으로, 소스 코드의 변경 이력을 관리하고 협업을 가능하게 하는 도구입니다.</p>
<h3 id="git의-주요-특징" tabindex="-1">Git의 주요 특징</h3>
<ul>
<li><strong>분산형 구조</strong>: 중앙 서버 없이도 각 클라이언트가 전체 데이터를 관리.</li>
<li><strong>효율성</strong>: 파일의 전체 스냅샷 대신 변경 사항만 저장.</li>
<li><strong>신뢰성</strong>: SHA-1 해시를 사용해 데이터 무결성을 보장.</li>
<li><strong>협업 지원</strong>: 여러 개발자가 동시에 작업 가능.</li>
</ul>
<h3 id="git의-장점" tabindex="-1">Git의 장점</h3>
<ol>
<li><strong>백업 기능</strong>: 작업 내역을 항상 저장소에 보관.</li>
<li><strong>협업 강화</strong>: 브랜치를 통해 각자 독립적으로 작업 가능.</li>
<li><strong>오픈소스</strong>: 무료로 사용 가능하며 커뮤니티가 활발.</li>
</ol>
<hr>
<h2 id="2-git-설치-및-설정" tabindex="-1">2. Git 설치 및 설정</h2>
<h3 id="git-설치" tabindex="-1">Git 설치</h3>
<ul>
<li>
<p><strong>Windows</strong></p>
<ol>
<li><a href="https://git-scm.com">Git 공식 웹사이트</a>에서 설치 파일 다운로드.</li>
<li>설치 중 기본 옵션 선택 추천.</li>
<li>설치 완료 후 <code>Git Bash</code> 실행.</li>
</ol>
</li>
<li>
<p><strong>Mac</strong></p>
<ol>
<li>Homebrew로 설치:<pre><code class="hljs language-bash">brew install git
</code></pre>
</li>
</ol>
</li>
<li>
<p><strong>Linux</strong></p>
<ol>
<li>
<p>Debian/Ubuntu:</p>
<pre><code class="hljs language-bash"><span class="hljs-built_in">sudo</span> apt update
<span class="hljs-built_in">sudo</span> apt install git
</code></pre>
</li>
<li>
<p>CentOS/Fedora:</p>
<pre><code class="hljs language-bash"><span class="hljs-built_in">sudo</span> dnf install git
</code></pre>
</li>
</ol>
</li>
</ul>
<h3 id="git-초기-설정" tabindex="-1">Git 초기 설정</h3>
<p>Git 설치 후, 사용자 정보와 에디터를 설정합니다.</p>
<ol>
<li>
<p>사용자 이름 설정:</p>
<pre><code class="hljs language-bash">git config --global user.name <span class="hljs-string">&quot;사용자이름&quot;</span>
</code></pre>
</li>
<li>
<p>사용자 이메일 설정:</p>
<pre><code class="hljs language-bash">git config --global user.email <span class="hljs-string">&quot;사용자이메일@example.com&quot;</span>
</code></pre>
</li>
<li>
<p>기본 에디터 설정 (예: VS Code):</p>
<pre><code class="hljs language-bash">git config --global core.editor <span class="hljs-string">&quot;code --wait&quot;</span>
</code></pre>
</li>
<li>
<p>설정 확인:</p>
<pre><code class="hljs language-bash">git config --list
</code></pre>
</li>
</ol>
